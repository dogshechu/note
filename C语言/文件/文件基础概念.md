### 1. 什么是文件？

文件是存储在外部介质（如硬盘、U盘）上的**数据集合**，每个文件都有一个唯一的**文件名**。

### 2. 文件的分类

- **文本文件**：内容以ASCII码字符形式存储
    
    - 可读性强，可用文本编辑器查看
        
    - 占用空间相对较大
        
    - 例如：.txt、.c、.h文件
        
- **二进制文件**：以二进制形式存储
    
    - 存储效率高，读写速度快
        
    - 不可直接阅读
        
    - 例如：.exe、.jpg、.dat文件
        

### 3. 文件指针（FILE指针）

c

```
FILE *fp;  // 声明文件指针
```

- 不直接指向文件内容，而是指向**文件信息区**
    
- 文件信息区包含：文件描述符、缓冲区、当前位置等信息
    
- 通过文件指针，程序可以访问和控制文件
## 总结

C语言的文件操作功能强大但需要细致处理。关键点包括：

1. **总是检查返回值**：文件操作可能失败
    
2. **及时关闭文件**：避免资源泄漏
    
3. **正确处理缓冲区**：确保数据及时写入
    
4. **理解文件指针位置**：避免意外覆盖
    
5. **选择合适的打开模式**：根据需求选择文本/二进制模式
